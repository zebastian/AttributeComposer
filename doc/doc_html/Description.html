Ds_Composer
Device Description


Revision: 1.0 - Author: K.Saintin


The device composer is used for the construction of a spectrum attribute
where datas come from numerical scalar attributes.
These attributes can be provided by any devices on the Tango database.
The user must give the name list of the attribute to read.
