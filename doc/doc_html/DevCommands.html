Device Commands Description


Revision: 1.1 - Author: K.Saintin



1 - Init

  • Description: This commands re-initialise a device.
    After an Init command executed on a device, the collecting process is taking in account any change on the AttributeNameList property
    Use this command to relaunch the processing if an attribute is avalaible whereas it was not before.
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    DEV_VOID     : none.
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON

     


2 - State

  • Description: This command gets the device state (stored in its device_state data member) and returns it to the caller.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    DEV_STATE     : State Code
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON

     


3 - Status

  • Description: This command gets the device status (stored in its device_status data member) and returns it to the caller.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    CONST_DEV_STRING     : Status descrition
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON

     


4 - Reset

  • Description: This commands re-initialise a device.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON

     


5 - GetAttributeNameForIndex

  • Description: Get the attributeName associated to the index of a spectrum
     
  • Argin:
    DEV_SHORT     : The index of the spectrum
     
  • Argout:
    DEV_STRING     : The name of the corresponding attribute
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON

     


6 - GetTangoQualities

  • Description: Get the existing tango qualities
     
  • Argin:
    DEV_VOID
  • Argout:
    DEVVAR_STRINGARRAY     : The name of the existing tango qualities
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON

     
     


7 - GetPriorityForQuality

  • Description: Get the priority number for a given tango quality
     
  • Argin:
    DEV_STRING     : The tango quality
     
  • Argout:
    DEV_SHORT     : The associated priority (defined in PriorityList property)
     
  • Command allowed for:
    • Tango::FAULT
    • Tango::ALARM
    • Tango::ON
