Device States Description


Revision: 1.0 - Author: K.Saintin



States
Names
Descriptions
STANDBY The device is ready to start the building process
RUNNING The device is reading the scalar attributes and building the spectrum
ALARM
  • One of the attributes is not reachable (device off), in this case associated the index of the spectrum will be set to NAN (Not a Number)
  • One of the attributes is not a numerical scalar type (String, spectrum), in this case the associated index of the spectrum will be set to NAN
  • One of the attributes are in ALARM
