States
Names
Descriptions
ON All the attributes are in the quality which the associated priority value is the lower.
By default VALID
MOVING All the attributes are in the CHANGING quality.
ALARM One of the attribute is in the quality which its priority value is immediately smaller than the higher value.
By default ALARM and WARNING
FAULT One of the attribute is in the quality which its associated priority value is the highest
By default INVALID and UNKNOWN
